# GOOGLE BOOK SEARCH

## Description

An app to search for and save books

![screenshoy](googlebooks.png)

## Installation

Run "npm install" in the terminal after cloning the project.

Run "npm start" in the terminal.

## Technologies Used

- React
- MongoDB
- Google Books API
- Bootstrap
- Node.js
- Express
